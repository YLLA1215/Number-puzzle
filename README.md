# Number-puzzle
数字华容道
# introduction-介绍  
这是可以在手机和电脑运行的“数字华容道”。相比于市面上其他的“数字华容道”来说，我写的“数字华容道”可能在操作上更顺滑，更符合真实的滑动操作体验。  
# Install-安装  
clone至本地即可。
# Instructions-使用方法  
将文件夹拖入VSCode后，运行index.html即可运行。  
后期可在服务器运行或制作成app至手机端使用。
